Content Editor

Use this page to edit project cards, project articles, and board members without touching the main website code. After editing, click Download Updated Content File. Then replace the old file at assets/data/site-content.js with the downloaded file and redeploy the website.

Projects

Each project becomes one card on the home page and one article page. The image must be inside assets/img/. Example: assets/img/project-9.jpg. For article text, write one paragraph per line.

Board of Officials

These names appear on the About Us page. Profile photos are not used, as requested earlier.
